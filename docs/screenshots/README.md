# Screenshots

[`../SCREENSHOTS.md`](../SCREENSHOTS.md) and the root `README.md`:

```
dashboard.png  agents.png  monitors.png  storage.png  wallboard.png
ticker.png  public-status.png  uptime.png  reports.png  notifications.png

```

Guidelines:

- **Redact** anything sensitive before capturing — real hostnames, IPs, customer
  names, emails, tokens. Use seeded/demo data or blur.
- PNG, ~800–1200px wide, **compressed** (e.g. TinyPNG / `pngquant`) to keep the repo
  light. Keep widths consistent.
- The images render automatically once committed.
